All About Bootstrap

Open Source

The webpage you see here is made using Bootstrap. Bootstrap is a popular open source front-end responsive CSS framework used in the production of websites. Bootstrap takes a mobile first approach. Bootstrap contains a collection of templates for website elements like buttons, cards, a navbar, and more.

Bootstrap Components library

Bootstrap contains a library of components. These components allow you to easily add elements to your website. Some components include buttons, the navbar, and more.

Bootstrap Document Creation

In order to use Bootstrap, it is required that you link many external files to a main html document. You will need to add a CSS stylesheet, some custom Bootstrap JavaScript, and a library for popovers called Popper. The HTML5 doctype, and the addition of meta tags are also required.

Bootstrap Components

Cards

Cards in bootstrap are basically flexible and responsive containers that you can add various content to. Cards can include Headers, footers, buttons, images, and more. Bootstrap grid can be used with cards to help with making the layout responsive.

Buttons

Bootstrap allows for the creation of custom buttons. Buttons are clickable elements on a webpage that allow you to submit forms, visit links, and more. Bootstraps buttons are similar to regular html buttons with the exception of special bootstrap classes. These buttons can be styled in many different ways. These include, changing the size with .btn-lg or .btn-sm. You can also add outlines, change their states, and more.

Nav Bar

Bootstrap allows you to easily create fully responsive navbars with drop-down menus which you can add to you website. Navbars contain links that allow you to easily navigate a website. Navbars require a .navbar with a .navbar-expand in order to be allowed to collapse smoothly when in mobile format.
